Konfigurasi Web App

Masukkan URL Google Apps Script Web App kamu agar scanner bisa terhubung ke Google Sheet.

Cara dapat URL ini:
1. Buka Apps Script → Deploy → New deployment
2. Type: Web App
3. Execute as: Me
4. Who has access: Anyone
5. Copy URL yang muncul, paste di atas

Penting: Setiap update Code.gs, buat deployment baru agar URL tetap valid.

Akses Kamera Ditolak

Chrome membutuhkan izin kamera. Ikuti langkah berikut:

Di Chrome Android:
Ketuk 🔒 di address bar → Izin Situs → Kamera → Izinkan → Reload

Di Chrome Desktop:
Klik 🔒/📷 di address bar → Camera → Allow → Reload

Pastikan halaman dibuka via HTTPS (GitHub Pages sudah HTTPS otomatis).
