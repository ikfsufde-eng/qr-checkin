Konfigurasi Web App

Masukkan URL Google Apps Script Web App agar scanner bisa terhubung ke Google Sheet.

Cara dapat URL ini:
1. Buka Apps Script → Deploy → New deployment
2. Type: Web App
3. Execute as: Me
4. Who has access: Anyone
5. Copy URL yang muncul (berakhiran /exec), paste di atas

Penting: Setiap update Code.gs, buat deployment baru agar perubahan aktif.
